Den här mappen innehåller:
- Images		: Mapp för bilder
- Info			: Mapp för information om programmet
- JsLibrary		: Mapp för inlånade javascript-filer
- StarshipGame		: Mapp för spelet Starship
- game.js 		: Programmet
- target.jpg		: En bild som används som exempel i spelet
- index.htm		: HTML-programmet som programmet körs från i webbläsare
- aktivitetsdiagram 	: Diagram som beskriver huvudragen av programflödet
